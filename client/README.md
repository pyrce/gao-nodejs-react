# Installation

## Serveur

Pour demarer le serveur, effectuez les commandes suivantes:

- cd serveur
- npm install
- nodemon

## Donnees

Pour créer les données:

- sequelize db:create
- sequelize db:migrate
- sequelize db:db:all

## Client

- cd client
- npm install
- npm start

l'applcation va ensuite se lancer sur localhost:8000
